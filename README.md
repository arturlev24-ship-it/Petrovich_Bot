# 🤖 Петрович — Весёлый чат-бот для Telegram

[![Python](https://img.shields.io/badge/Python-3.8%2B-blue)](https://python.org)
[![aiogram](https://img.shields.io/badge/aiogram-3.x-green)](https://docs.aiogram.dev/)
[![License](https://img.shields.io/badge/License-MIT-yellow)](LICENSE)

**Петрович** — это лёгкий и прикольный бот для групповых чатов Telegram, который оживляет беседу, реагирует на слова-триггеры, приветствует новых участников и поднимает настроение!

---

## ✨ Возможности

| Функция | Описание |
|---------|----------|
| 🎯 **300+ реакций** | Бот реагирует на слова в 25+ категориях |
| 👋 **Приветствия** | Встречает новых участников забавными фразами |
| 🏆 **Система кармы** | Начисляет очки за активность, ведёт ТОП |
| 💬 **Умные ответы** | Понимает контекст: еда, работа, погода, игры, музыка и др. |
| 😄 **Развлечения** | Шутки, факты, комплименты по командам |
| 💾 **Сохранение** | Карма сохраняется в JSON каждые 10 минут |
| 🛡️ **Защита** | Обработка ошибок, защита от флуда, игнор ботов |
| 📊 **Статистика** | Показывает время работы и количество сообщений |
